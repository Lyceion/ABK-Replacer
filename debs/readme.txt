*** WORKS ONLY WITH MW ***

Use command prompt to run abk_insert.
=== Drive:/Folder/cd abk_insert.exe ===

command :
ABK_insert [any MW format.abk] [16 bit mono.wav] [num of sample (car tranny = 1-2)(eng mb spu = 1-8)]

example :
ABK_insert 4CHANNEL.abk SAMPLE_WAV00.wav 1
ABK_insert 4CHANNELm.abk SAMPLE_WAV01.wav 2
ABK_insert 4CHANNELmm.abk SAMPLE_WAV02.wav 3
ABK_insert 4CHANNELmmm.abk SAMPLE_WAV03.wav 4
